## 4. Аналитика бренда (на основе фирменных материалов ресторана)

### 4.1. Фирменный стиль (по материалам в папках проекта)

Заказчик передал фотографии и фирменные макеты меню в четырёх папках: `Меню кухня/`, `Меню бар/`, `Детское меню/`, `Севастопольские сезоны/`. На их основе выведена следующая визуальная система:

**Логотип** (см. `Детское меню/WVTzF_23U7hvLOq6t5liQSYy28JuuLVVYPT9F-0fVMN2wG0qhb7BFiTx5WVCNAVK6pqJYWAaG-OVQ4ig02qh6cZG.jpg`):
- Wordmark «МОМО» — крупный геометрический гротеск, обрамлённый горизонтальными линиями сверху и снизу.
- Таглайн: «семейный ресторан с видом на море» — тонкий sans под основным wordmark.
- Подпись: *since 2013*.

**Ключевой визуал бренда** — ботаническая иллюстрация лотосов в винтажной графической манере (пыльно-розовые цветы, мятно-зелёные листья, кремовые акценты). Это можно и нужно использовать как декоративный «герой» на сайте — в углах секций, на заглавных экранах.

**Палитра бренда MOMO** (читается с фирменных макетов и реальных фото интерьера):
- Основной фон — **пыльно-розовый/peach** (rose dust): около `#F2C9C6` — `#E8BCB8`.
- Акцент — **мятно-зелёный/sage**: около `#A9C9B8` — `#88B5A0`.
- Фоны для бар-меню — **сливочный/cream** (`Меню бар/19JEdNQcz6W992DwZLwd...`) и **глубокий бордо/велюр** для «согревающих напитков».
- Детское меню — **бирюзовый/teal** (`#5BB8B5`) на белом, с playful иллюстрациями.

⚠️ **Севастопольские сезоны — внешний фестиваль, не часть бренда MOMO.** Это городской гастро-проект, организованный sevastravel.ru, в котором MOMO участвует от сезона к сезону. **Палитра, шрифты и обложки этого фестиваля принадлежат фестивалю, не ресторану** — и они меняются с каждой сессией (весна/лето/осень/зима). Поэтому: не использовать насыщенный розовый из весенней сессии 2026 как «один из режимов MOMO». На сайте фестиваль показывается как **участие** (бейдж, ссылка на sevastravel.ru, сменное сезонное меню в палитре MOMO), а не как фирменный цветовой блок.

**Свет и кадрирование** (фото блюд):
- **Кухня и бар**: дневной свет, светлый фон (лён, светлое дерево, кремовая поверхность). Студийная, но не глянцевая подача — блюдо как герой (см. `Меню кухня/-_mO0NQRF...`).
- **Согревающие напитки**: на бордовом велюре, тёплый вечерний свет, золотые блюдца — для «уютной» части меню (`Меню бар/5Hz7aBxV...`).
- **Фестивальные фото** на розовом атласе (папка `Севастопольские сезоны/`) — оформление **фестиваля**, не MOMO. Для лендинга использовать только содержимое (блюда, описания, цены), не визуальную манеру.

Иными словами, у бренда **две атмосферы под одной крышей**: светлая дневная (основной зал, кухня, детское) и тёплая вечерняя (бар, согревающие). Сайт должен показать обе.

### 4.2. Tone of Voice
Из самопрезентации ресторана: **«семейный ресторан в центре Севастополя с красивейшим видом на море»**, кухня **«построена на традициях европейской и азиатской кухни с использованием сезонных локальных продуктов»**, миссия — **«подарить незабываемые эмоции»**.

**Голос бренда:**
- Тёплый, гостеприимный, на «вы», но не официальный.
- Через атмосферу и эмоции, а не через скидки и «акции».
- Подчёркивает **локальность** (крымские продукты, севастопольская бухта) и **семейность**.
- Без пафоса, без англицизмов в духе chef’s table / signature menu.
- Короткие фразы, конкретика вместо абстракций.

**Можно:** «свежий пеленгас», «вид на бухту», «джаз и медленный ужин», «детская комната с раскрасками».
**Нельзя:** «премиальный гастрономический опыт», «инновационная концепция», «не имеет аналогов», «вкус, который вас покорит».

## 5. Дизайн-система

### 5.1. Палитра (синхронизирована с фирстилем ресторана)
Бриф: белый, бежевый, розовый, светло-серый. **Чёрный — исключён.** Базовый текст — глубокий тёмно-серый, не #000. По итогам анализа фирменных материалов **розовый и мятный — основные бренд-цвета, не акценты**.

| Роль | HEX | Назначение |
|---|---|---|
| `--bg` фон | `#FBF8F4` | Тёплый off-white, основной фон сайта |
| `--surface` | `#FFFFFF` | Карточки блюд, всплывающие элементы |
| `--rose-soft` | `#F2C9C6` | Бренд-розовый (фоны секций «О нас», «Для семей») |
| `--rose` | `#E8BCB8` | Бренд-розовый насыщенный (заглавные плашки) |
| `--rose-deep` | `#C97B70` | Акцент-роза (CTA, ссылки) |
| `--sage` | `#A9C9B8` | Бренд-мятный (декор, иконки, бордеры) |
| `--sage-deep` | `#88B5A0` | Мятный акцент (hover, активные состояния) |
| `--cream` | `#F5EFE3` | Кремовый фон (как в бар-меню) |
| `--beige` | `#EFE4D2` | Секционные блоки, разделители |

| `--ink-soft` | `#6B655E` | Вторичный текст, подписи |
| `--teal-kids` | `#5BB8B5` | Только для блока «Детское меню» — повторяет фирстиль детского буклета |
| `--velvet` | `#5C3A4A` | Только для блока «Согревающие напитки» / вечерний бар (тёплый бордо-велюр) |

Светотень: длинные мягкие тени, без резких контрастов. Никаких чёрных подложек.

### 5.2. Типографика (трёхшрифтовая система — под три «голоса» бренда)

Двухшрифтовая система MOMO:

- **Logo / wordmark MOMO:** `Futura` / `Avenir Next Demi` / `Geist Sans Semibold` — повторяет фирменный wordmark с горизонтальными линиями над и под буквами.
- **Заголовки H1/H2 (основные секции):** `Cormorant Garamond` — серьёзный антиквенный сериф, тёплый и «семейный».
- **Тело:** `Inter` (или `Manrope`) — нейтральный, отлично читается на всех фонах.
- **Микро-акценты, ингредиенты в карточках блюд:** тот же `Inter` light + всё прописными с трекингом — повторяет манеру оригинальных макетов меню.

> **Не путать с шрифтами фестиваля.** В макетах «Севастопольских сезонов» используется бруталистский ретро-display-сериф (`Cubano` / `Cooper Std Black`) — это шрифт фестиваля sevastravel.ru, не MOMO. На лендинге MOMO для секции сезонного меню используем **типографику MOMO**, не фестиваля.

Размеры (desktop):
- H1: 64–80px, line-height 1.05
- H2: 40–48px
- H3: 24–28px
- Body: 17–18px, line-height 1.6
- Small: 14px

### 5.3. Сетка и ритм
- Контейнер: 1200px max-width, отступы 24/48/96px.
- Сетка 12 колонок (desktop), 6 (tablet), 4 (mobile).
- Mobile-first. Карточки меню — 1 колонка на mobile, 2 на tablet, 3 на desktop.
- Радиусы скруглений: 16–24px у карточек, 999px у кнопок (мягкость, отсутствие острых углов — поддерживает «семейный»).

### 5.4. Иконки и иллюстрации
- Иконки — outline, 1.5px, скруглённые концы.
- Уместны мелкие декоративные иллюстрации: персик (привет названию), волна, лимон, бокал, листок.
- Никаких 3D-рендеров, никакого «glassmorphism».

### 5.5. Анимации
- Только мягкие: fade-in блюд при скролле, плавное появление текста, медленные ken-burns на фоновых фото моря.
- Никаких параллаксов «как в казино», никаких летающих частиц.

### 6.3. Севастопольские сезоны

**Что это:** собственный блок MOMO с **текущим сезонным сетом** (4 блюда + 4 коктейля). В нашей реализации блок **универсальный и переключаемый**:

- Когда MOMO участвует в фестивале «Севастопольские сезоны» — внутри блока появляется **бейдж «участник фестиваля Севастопольские сезоны → sevastravel.ru»**, а сет соответствует фестивальной программе сезона.
- Когда сессия фестиваля закончена или MOMO не участвует — бейдж убирается, остаётся **собственное сезонное меню MOMO**.

Это решает риск-кейс #4.6 из [RISKS.md](RISKS.md): фестиваль — внешний проект (организатор sevastravel.ru), его палитра и шрифты меняются от сезона к сезону и принадлежат фестивалю, не MOMO. Поэтому **секция оформляется в фирменной палитре MOMO** (см. §5.1), а не в фестивальной розовой.

## 7. Бронирование

### 7.1. Главный канал — звонок администратору

На MVP-этапе **единственный канал бронирования — телефон**. Telegram-бот и интеграции с системами (Restoplace, YClients) — отложены в V2/V3.

- Номер: **+7 (978) 855-75-54** (телефон администратора зала).
- Кнопка CTA на сайте: **«Забронировать стол · +7 978 855-75-54»** → `tel:+79788557554`.
- На мобильных — один тап открывает звонилку. На десктопе — крупно отображённый номер для набора руками + системный `tel:` handler для тех, у кого настроен Skype / FaceTime / Линфон.

**Почему так:**
- В Севастополе и Крыму у Telegram периодически плохое соединение с мобильными операторами — звонок надёжнее.
- У части аудитории 45+ нет Telegram-аккаунта.
- Меньше инфраструктуры на старте — меньше точек отказа.
- Конверсия `tel:` CTA на ресторанных лендингах обычно 6–12% от трафика.

### 7.2. Резервный канал — форма «Перезвоните мне»

Не все готовы звонить сами (стесняются, на работе, после рабочих часов ресторана). Поэтому рядом с главным CTA — кнопка **«Перезвоните мне»** с короткой формой:

- Поле «Имя» (необязательное).
- Поле «Телефон» (обязательное, маска `+7 (___) ___-__-__`).
- Поле «Когда удобно перезвонить» — селект на 3 опции: «Сейчас», «В течение часа», «Сегодня вечером».
- Чекбокс согласия на обработку ПД (152-ФЗ).
- Кнопка отправки.

**Куда приходит заявка:**
- Email на адрес администратора + дублирующее SMS на номер дежурного.
- Опционально (V2) — в чат-группу администрации (Telegram / WhatsApp / Bitrix24 — по выбору заказчика).

**SLA:** обратный звонок в течение 15 минут в рабочее время; на следующий день, если форма отправлена ночью.

Под формой «Перезвоните мне»:

### 7.5. Telegram-бот / Restoplace / прочее — что делаем в V2 и позже

Если объём звонков вырастет и администраторы начнут не успевать — на V2 рассматриваем:
- **[Restoplace](https://restoplace.cc/)** виджет — электронная книга бронирований с автоматическим подбором свободного времени. От 2 020 ₽/мес.
- **Свой Telegram-бот** — кастомный UX, но требует разработки (~30 000 ₽ единоразово) и постоянного сопровождения.
- **WhatsApp-кнопка** — `wa.me/79788557554` — для тех, кому удобнее текст, чем голос.

На MVP-этапе ни одно из этих решений **не нужно**. Главное правило — не плодить каналы коммуникации, пока команда не освоилась с одним.

## 9. Технические требования

### 9.1. Стек (предложение)
- **Фронтенд:** Next.js (App Router) + TypeScript + Tailwind CSS — быстро, SEO-friendly, легко поддерживать.
- **Деплой:** Vercel (бесплатный тариф покрывает с запасом).
- **Аналитика:** Yandex.Metrica (вебвизор для понимания поведения), события на клик по CTA «Бронь» и «Маршрут».
- **Карта:** Яндекс.Карты JS API (встраиваемый виджет).
- **Изображения:** Next/Image, WebP/AVIF, lazy-load.

### 9.2. Производительность
- Lighthouse: Performance ≥ 90, Accessibility ≥ 95, SEO ≥ 95.
- LCP < 2.5s на 4G (Hero-изображение оптимизировано, ≤ 250kb).
- CLS < 0.1.

### 9.3. SEO
- Title: `МОМО — семейный ресторан с видом на бухту в Севастополе, пр. Нахимова, 2А`
- Description: одна фраза о виде, кухне и адресе.
- Schema.org `Restaurant` с адресом, телефоном, часами, координатами, ценовой категорией.
- OpenGraph картинка (hero-кадр).
- `sitemap.xml`, `robots.txt`, canonical.

### 9.4. Адаптивность
- Mobile-first.
- Breakpoints: 360 / 768 / 1024 / 1440.
- Все CTA — большие, в зоне большого пальца на mobile.
- Меню на mobile — карточки в одну колонку, табы — горизонтальный скролл.

### 9.5. Доступность
- Контрастность текста к фону ≥ 4.5:1 (особенно на бежевом фоне!).
- Все интерактивные элементы — с focus-state.
- Alt у всех изображений.
- Семантические теги.
